📚 Description du projet

LireMonde est une plateforme web interactive développée pour la startup fictive ReadSphere, spécialisée dans la culture numérique.

L’objectif du projet est de moderniser l’expérience de lecture en ligne en proposant une bibliothèque numérique dynamique, intuitive et responsive.

L’application permet aux utilisateurs de :

consulter des livres dynamiquement
rechercher des ouvrages en temps réel
filtrer les livres par catégorie
afficher les détails d’un livre dans une modale
gérer une liste personnelle “À lire”
administrer les livres grâce à un tableau de bord CRUD

Le projet fonctionne avec une API REST simulée via JSON Server et un fichier db.json.

🔌 API & JSON Server

Le projet utilise JSON Server pour simuler une API REST locale.

Installation
npm install -g json-server
Lancer le serveur
json-server --watch db.json --port 3000

API disponible sur :

http://localhost:3000/books
